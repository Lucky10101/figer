# figer
基于 PyTorch 搭建 100 类手指/掌静脉二值图像识别系统，完成数据划分、ResNet18 baseline 训练、梯度扰动生成、增强训练与模型对比评估。设计标签保持型 FGSM / binary bit-flip 扰动方法，通过分类器梯度选择关键像素翻转，并引入 keep rate、像素变化比例、L1/L2 距离、前景比例差异等指标评估扰动有效性。最终形成从 baseline 训练、扰动生成、增强数据集构建、再训练到结果对比的完整实验闭环。Baseline 在 100 类身份识别任务上达到 Test Acc 0.86；为后续引入结构约束、PGD、形变场和可学习扰动网络奠定基础。
